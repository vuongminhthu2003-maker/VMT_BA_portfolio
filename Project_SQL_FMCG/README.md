# FMCG Sales Analytics (Favorita Dataset)

## Project Overview
This project analyzes FMCG retail sales data from the Favorita grocery chain to understand sales performance across product categories and stores.

The goal is to identify key drivers of sales and support better inventory planning and promotion strategies.

## Business Questions
- Which product categories generate the most sales?
- Which stores perform best?
- How do promotions affect product sales?
- How do sales change over time?

## Dataset
Favorita Grocery Sales Forecasting dataset from Kaggle.

Main tables used:
- train (daily sales data)
- items (product categories)
- stores (store information)
- transactions (store transaction counts)

## Tools
- SQL for data analysis
- Power BI for dashboard visualization
